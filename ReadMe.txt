Project #1: Artillery Game
Due: 2022-10-05

Build and run through Visual Studio as normal. If ran directly through the exe it will not run.

Goal: Hit the enemy(red) tank by utilizing the 5 different artillery shots as listed below.

User inputs: 
W - Aim forward
S - Aim backward
A - Aim left
D - Aim right
E - Aim up
Q - Aim down
X - Fire
1 - Normal shot
2 - Drop shot
3 - Triple shot
4 - Railgun shot
5 - Bounce shot
N - Start new game(must've hit enemy)
